# jwp-shopping-order

# API

| 기능 | 메소드 | url | 구현 여부 |
| --- | --- | --- |-------|
| 주문하기 | POST | ~/orders | o     |
| 주문 목록 조회 | GET | ~/orders | o     |
| 주문 상세 보기 | GET | ~/orders/{id} | o     |
| 주문 확정 | PATCH | ~/orders/{id}/confirm | o     |
| 주문 취소 | DELETE | ~/orders/{id} | o     |
| 유저 쿠폰 발급 | POST | ~/users/coupons | o     |
| 모든 쿠폰 조회 | GET | ~/coupons | o     |
| 유저 쿠폰 조회 | GET | ~/users/coupons | o     |

- **[POST] 주문하기 (~/orders)**

  ### **Request**

    - header

    ```
    Authorization: Basic ...
    ```

    - body

    ```json
    {
            "selectCartIds" : [1, 2, 3],
            "couponId" : 1
    }
    ```

  ### **Response**

    - status: 201(Created)
    - header

        ```
        Location: orders/{id}
        ```


- **[GET] 주문 목록 가져오기 (~/orders)**

  ### **Request**

    - header

    ```

  ### **Response**

    - body

          {

               "id" : 2,
               "name" : "피자",

      {

    ```

    ```

        "coupon" :{

    		"discountType" : "percentage",

    	}
    }
    ```


- **[DELETE] 주문 취소 (~/orders/{id})**

  ### **Request**

    - header

    ```
    Authorization: Basic ...
    ```

  ### **Response**

    - status: 204(No Content)
